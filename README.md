# 🎬 Movie Collection

Aplicação web desenvolvida em equipe durante o curso de Desenvolvimento Fullstack do SENAI para pesquisa e visualização de filmes utilizando a The Movie Database (TMDB) API.

## 📌 Funcionalidades

- 🔎 Pesquisa de filmes por título.
- 🎥 Visualização de informações dos filmes encontrados.
- ❤️ Gerenciamento de filmes favoritos.
- 📱 Interface responsiva para diferentes dispositivos.

## 🚀 Tecnologias

- React
- JavaScript
- Vite
- HTML5
- CSS3
- Axios
- TMDB API
- Git e GitHub

## 💻 Minha participação

- Desenvolvimento de funcionalidades utilizando React.
- Integração da aplicação com a TMDB API para busca de filmes.
- Implementação e ajustes na interface da aplicação.
- Testes das funcionalidades durante o desenvolvimento.
- Versionamento do código utilizando Git e GitHub.
